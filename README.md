# node-install
一款适配sspanel的后端对接的工具箱

### 使用方法
```shell
wget -N --no-check-certificate "https://raw.githubusercontent.com/bobkjl/node-install/master/node-install.sh" && bash node-install.sh
```

### 功能说明
- 一键安装Rico付费版/免费版v2ray后端
- 一键安装ssr后端
- 一键安装TCP加速
- 一键服务器测速
- 一键服务器性能测试
